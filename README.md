# MYFATE 命理文档

> 八字命理知识库 - [myfate.org](https://myfate.org) 的智慧源泉

这是 MYFATE 智命平台的命理知识文档，涵盖八字四柱命理的核心概念与计算方法。

**在线阅读**: [docs.myfate.org](https://docs.myfate.org)

## 内容

### 基础知识
- 阴阳 - 对立统一的基本原理
- 五行 - 木火土金水的相生相克
- 天干 - 甲乙丙丁戊己庚辛壬癸
- 地支 - 子丑寅卯辰巳午未申酉戌亥
- 干支 - 六十甲子纪年系统
- 八字 - 四柱命理架构

### 进阶概念
- 旺相休囚死 - 五行季节强弱
- 十神 - 干支关系分类
- 生耗值 - 日主强弱计算
- 特殊格局 - 命理格局判断

## 欢迎贡献

这是一个开源的命理知识库，**欢迎任何人修改和完善**。

如果你发现错误、有更好的解释、或想添加新内容：

1. Fork 这个仓库
2. 修改或添加 `src/content/docs/` 中的文档
3. 提交 Pull Request

无论是修正一个错字，还是贡献一整篇文章，我们都非常感谢。
