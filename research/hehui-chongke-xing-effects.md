# 合会冲克刑对五行力量的影响研究报告

## 概述

在八字命理中，天干地支之间的各种关系会显著影响五行能量的计算。本报告总结了各种关系及其对五行力量的具体影响。

---

## 一、力量等级排序

根据传统命理学，各种地支关系的力量从强到弱排列：

```
三会 > 三合 > 相冲 > 相刑 > 六合 > 半合 > 相害 > 相破
```

---

## 二、天干关系

### 2.1 天干五合 (Heavenly Stem Combinations)

| 组合 | 化神 | 条件 |
|------|------|------|
| 甲己合 | 化土 | 辰戌丑未月 |
| 乙庚合 | 化金 | 申酉月 |
| 丙辛合 | 化水 | 亥子月 |
| 丁壬合 | 化木 | 寅卯月 |
| 戊癸合 | 化火 | 巳午月 |

#### 合化成功条件
1. **化神得令** - 化出的五行在月令得旺相
2. **无妒合** - 没有第三个天干来争合
3. **无克制** - 化神不受克制
4. **紧贴相合** - 两干相邻

#### 力量变化
- **合化成功**: 原来两干的五行能量转化为化神五行，化神力量增强
- **合而不化（合绊）**: 双方力量都减弱约30-50%，相互牵制

### 2.2 天干相克

天干相克按正常五行相克规则，克者消耗能量，被克者减少能量。

## 三、地支关系

### 3.1 地支六合 (Six Combinations)

| 组合 | 化神 | 说明 |
|------|------|------|
| 子丑合 | 化土 | 水土相合 |
| 寅亥合 | 化木 | 木水相合 |
| 卯戌合 | 化火 | 木土相合 |
| 辰酉合 | 化金 | 土金相合 |
| 巳申合 | 化水 | 火金相合 |
| 午未合 | 化火/土 | 火土相合 |

#### 力量变化
- **合化成功**: 双方五行转化为化神五行
- **合而不化**: 双方力量减弱，但比天干合绊减弱程度小
- **基础力量值**: 约36度

### 3.2 地支三合局 (Triangular Combinations)

| 三合局 | 组成 | 化神 | 力量 |
|--------|------|------|------|
| 申子辰 | 申+子+辰 | 水局 | 约60度 |
| 亥卯未 | 亥+卯+未 | 木局 | 约60度 |
| 寅午戌 | 寅+午+戌 | 火局 | 约60度 |
| 巳酉丑 | 巳+酉+丑 | 金局 | 约60度 |

#### 半合局
- **生旺半合**: 如寅午（火局之生旺），力量较强
- **旺墓半合**: 如午戌（火局之旺墓），力量次之
- **力量值**: 约30-40度

### 3.3 地支三会方 (Directional Combinations)

| 三会 | 组成 | 化神 | 力量 |
|------|------|------|------|
| 寅卯辰 | 东方 | 木 | 约72度 |
| 巳午未 | 南方 | 火 | 约72度 |
| 申酉戌 | 西方 | 金 | 约72度 |
| 亥子丑 | 北方 | 水 | 约72度 |

三会方的力量最强，因为是同一方位的三个地支完全会合。

### 3.4 地支六冲 (Six Clashes)

| 冲 | 组成 | 影响 |
|----|------|------|
| 子午冲 | 子↔午 | 水火交战 |
| 丑未冲 | 丑↔未 | 土土相冲 |
| 寅申冲 | 寅↔申 | 木金交战 |
| 卯酉冲 | 卯↔酉 | 木金交战 |
| 辰戌冲 | 辰↔戌 | 土土相冲 |
| 巳亥冲 | 巳↔亥 | 火水交战 |

#### 力量变化
- 相冲双方力量都会减弱
- 旺者减力较小，衰者减力较大
- 可能导致藏干透出或被冲散
- **力量值**: 约减少40-60%

### 3.5 地支三刑 (Three Punishments)

| 刑类 | 组成 | 说明 |
|------|------|------|
| 无恩之刑 | 寅刑巳，巳刑申，申刑寅 | 寅巳申三刑 |
| 恃势之刑 | 丑刑戌，戌刑未，未刑丑 | 丑戌未三刑 |
| 无礼之刑 | 子刑卯，卯刑子 | 子卯相刑 |
| 自刑 | 辰辰、午午、酉酉、亥亥 | 同支自刑 |

#### 力量变化
- 相刑会使双方力量减弱
- 同时可能引动藏干
- **力量值**: 约减少20-40%

### 3.6 地支六害 (Six Harms)

| 害 | 组成 | 说明 |
|----|------|------|
| 子未害 | 子↔未 | 六合被冲破 |
| 丑午害 | 丑↔午 | 六合被冲破 |
| 寅巳害 | 寅↔巳 | 六合被冲破 |
| 卯辰害 | 卯↔辰 | 六合被冲破 |
| 申亥害 | 申↔亥 | 六合被冲破 |
| 酉戌害 | 酉↔戌 | 六合被冲破 |

#### 力量变化
- 相害的力量影响较小
- 主要表现为暗中损耗
- **力量值**: 约减少10-20%

### 3.7 地支相破

| 破 | 组成 |
|----|------|
| 子酉破 | 子↔酉 |
| 午卯破 | 午↔卯 |
| 寅亥破 | 寅↔亥 |
| 巳申破 | 巳↔申 |
| 辰丑破 | 辰↔丑 |
| 戌未破 | 戌↔未 |

相破力量最弱，一般不作为主要因素考虑。

## 五、实现建议

### 5.1 计算顺序

1. 先计算基础五行力量
2. 检查三会方（最高优先级）
3. 检查三合局
4. 检查六合
5. 检查相冲（可能破坏合局）
6. 检查相刑
7. 检查相害

### 5.2 冲破规则

- 相冲可以破坏六合
- 相冲可以破坏半合
- 三合局/三会方较难被冲破（需要多个冲）

### 5.3 合化条件判断

需要检查：
1. 是否得月令
2. 是否有妒合
3. 是否被克制
4. 是否紧贴
