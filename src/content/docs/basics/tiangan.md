天干共有十个，分为五组，每组一阴一阳，对应五行。

## 十天干

| 天干 | 五行 | 阴阳 |
|-----|-----|-----|
| 甲 | 木 | 阳 |
| 乙 | 木 | 阴 |
| 丙 | 火 | 阳 |
| 丁 | 火 | 阴 |
| 戊 | 土 | 阳 |
| 己 | 土 | 阴 |
| 庚 | 金 | 阳 |
| 辛 | 金 | 阴 |
| 壬 | 水 | 阳 |
| 癸 | 水 | 阴 |

## 天干特性

### 阳干
- **甲木**：参天大树，刚直向上
- **丙火**：太阳之火，光明普照
- **戊土**：高山大地，厚重稳固
- **庚金**：刀剑之金，刚硬锋利
- **壬水**：江河大海，奔腾不息

### 阴干
- **乙木**：花草藤蔓，柔韧灵活
- **丁火**：灯烛之火，温暖细腻
- **己土**：田园沃土，滋养万物
- **辛金**：珠玉首饰，精致内敛
- **癸水**：雨露泉水，滋润渗透
