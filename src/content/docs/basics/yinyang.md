---
title: 阴阳
description: 阴阳的基本概念与属性
---

阴阳是中国哲学的核心概念，代表宇宙万物对立统一的两种基本属性。

## 基本属性

| 阳 | 阴 |
|---|---|
| 明亮 | 黑暗 |
| 热 | 冷 |
| 动 | 静 |
| 刚 | 柔 |
| 外 | 内 |
| 上 | 下 |

## 在八字中的应用

天干和地支各有阴阳之分：

### 天干阴阳

| 阳干 | 阴干 |
|-----|-----|
| 甲（木） | 乙（木） |
| 丙（火） | 丁（火） |
| 戊（土） | 己（土） |
| 庚（金） | 辛（金） |
| 壬（水） | 癸（水） |

### 地支阴阳

| 阳支 | 阴支 |
|-----|-----|
| 子、寅、辰、午、申、戌 | 丑、卯、巳、未、酉、亥 |

## 阴阳互根

阴中有阳，阳中有阴。阴阳相互依存，相互转化。
