## 藏干

每个地支内含有1-3个藏干，比例不同。

:::caution[注意]
藏干比例因流派不同而有差异，以下为本系统使用的参数值。
:::

| 地支 | 本气 | 中气 | 余气 |
|-----|-----|-----|-----|
| 子 | 癸(100%) | - | - |
| 丑 | 己(50%) | 癸(30%) | 辛(20%) |
| 寅 | 甲(60%) | 丙(30%) | 戊(10%) |
| 卯 | 乙(100%) | - | - |
| 辰 | 戊(50%) | 乙(30%) | 癸(20%) |
| 巳 | 丙(60%) | 戊(30%) | 庚(10%) |
| 午 | 丁(50%) | 己(50%) | - |
| 未 | 己(50%) | 丁(30%) | 乙(20%) |
| 申 | 庚(60%) | 壬(30%) | 戊(10%) |
| 酉 | 辛(100%) | - | - |
| 戌 | 戊(50%) | 辛(30%) | 丁(20%) |
| 亥 | 壬(70%) | 甲(30%) | - |

:::note
藏干比例用于计算[生耗值](/advanced/shenghao/)，是判断日主强弱的重要因素。
:::

## 三合局

地支三合可化成某一五行：

- **申子辰** 合水局
- **寅午戌** 合火局
- **巳酉丑** 合金局
- **亥卯未** 合木局

## 六冲

相隔六位的地支相冲：

子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
