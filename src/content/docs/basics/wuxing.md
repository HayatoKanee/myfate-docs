---
title: 五行
description: 五行的基本概念、相生相克关系
---

五行是中国古代哲学的核心概念，代表能量转化的五种状态：**木、火、土、金、水**。

## 五行属性

| 五行 | 方位 | 季节 | 颜色 |
|-----|-----|-----|-----|
| 木 | 东 | 春 | 绿 |
| 火 | 南 | 夏 | 红 |
| 土 | 中 | 长夏 | 黄 |
| 金 | 西 | 秋 | 白 |
| 水 | 北 | 冬 | 黑 |

## 相生关系

五行相生，循环不息：

```
木 → 火 → 土 → 金 → 水 → 木
```

- 木生火（木燃烧生火）
- 火生土（火燃尽成灰土）
- 土生金（土中蕴藏金属）
- 金生水（金属熔化成液）
- 水生木（水滋养树木）

## 相克关系

五行相克，相互制约：

```
木 → 土 → 水 → 火 → 金 → 木
```

- 木克土（树根破土）
- 土克水（土挡水流）
- 水克火（水灭火焰）
- 火克金（火熔金属）
- 金克木（金斧伐木）
